This directory contains the Rust implementation of STEF.

# Crates

- `stef-core`: Core STEF encoding/decoding primitives and schema support.
- `stef-grpc`: gRPC transport built on top of `stef-core`.
- `stef-otel`: Generated OpenTelemetry/STEF bindings.

# Prerequisites

- Install a recent Rust toolchain (`rustup`, `cargo`).
- Install [Go 1.25 or newer](https://go.dev/doc/install) for cross-language generation and interop checks.

# Building And Testing

To build and test all Rust crates:

```bash
cd rust
cargo test --workspace
```

To regenerate Otel/STEF bindings for Go, Java, and Rust:

```bash
cd go/otel
make generate
```

To run generated-schema tests from `stefc` (Go, Java, and Rust generation):

```bash
cd stefc
make test
```

To run Rust-Go interop tests generated by `stefc` tooling:

```bash
cd stefc/generator/testdata/out/all_features.stef/rust
cargo test -- --ignored
```
